# Bar Inventory & Sales Analysis (Excel)

## Project Overview
This project focuses on managing bar stock levels and analyzing guest sales patterns at Catamaran Brewing Company. The goal was to reduce wastage and optimize inventory based on demand.

## Tools Used
* **MS Excel:** Pivot Tables, VLOOKUP, and Data Formatting.
* **Logic:** Conditional Formatting for 'Low Stock' alerts.

## Key Insights & Contributions
* **Inventory Management:** Tracked daily stock counts and automated re-order alerts when stock fell below 10%.
* **Sales Analysis:** Identified that weekend dinner hours have 30% higher sales, helping in better staff planning.
* **Waste Reduction:** Reduced liquor wastage by 12% through accurate inventory reconciliation.

## How to use this project?
1. Open the `Inventory_Tracker.xlsx` file.
2. Check the 'Dashboard' sheet for visual charts.
3. Review the 'Stock' sheet for automated formulas.
